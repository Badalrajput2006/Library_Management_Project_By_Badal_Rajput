# 📚 Library Management System

A Library Management System is a web-based application designed to manage library operations efficiently. It helps in maintaining records of books, users, and transactions like issue and return of books.

---

## 🚀 Features

- 🔐 User Authentication (Login / Logout)
- 📖 Add, Update, Delete Books
- 👤 Manage Users
- 📕 Issue Books
- 📗 Return Books
- 🔍 Search Books
- 📊 View Available and Issued Books

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JSP
- **Backend:** Java (Servlets)
- **Database:** MySQL
- **Server:** Apache Tomcat
- **IDE:** Eclipse
